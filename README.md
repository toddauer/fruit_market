Each file is a build of the previous file or contains like-elements where a different concept was explored.
This code was largely generated by Claude AI and used as a learning tool. Anyone who finds this repo can also learn
from these common e-commerce themes for HTML, CSS, and JavaScript.

All projects kept as single file launch to ensure all elements were compatible and code runs without issue.

File Layout:
  1. Fruit Filter. This is the base. It has tiles for each fruit along with metadata tags that describe each fruit. Only one keyword may be used in the search field.
  2. Fruit Filter with Sub Filter Elements. Builds on fruit filter to allow multiple keyword search results. Searches are appended below the search field and can be removed.
  3. Fruit Filter Side Menu Elements. Changes the overall layout and filter selection type. Instead of typed keywords, you can use check boxes as filter criteria.
  4. Fruit Filter With Sub Config and Cart. Builds on Fruit Filter Side Menu Elements, but adds that iconic sub-menu configuration before adding to cart.

Again, these are for learning purposes and do not qualify as ready-for-use in a production site.
